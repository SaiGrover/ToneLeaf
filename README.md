# Toneleaf

Toneleaf is a calm, privacy-focused web application that helps people understand the emotional tone behind written language. It presents sentiment as clear positive, neutral, negative, or mixed signals and can also highlight wording associated with emotional distress.

The experience supports typed text, text documents, voice transcripts, and social-post content. Results include an easy-to-read signal breakdown, contextual cues, copyable summaries, and a private analysis history stored in the browser.

Designed with a soft pink-and-blue visual system, responsive layouts, and light and dark themes, Toneleaf makes emotional language analysis approachable across desktop and mobile devices. All analysis runs locally in the browser, so submitted text is not sent to an external service.

Toneleaf is intended for reflection and language awareness. Its distress check is not a medical diagnosis or a replacement for professional support.
